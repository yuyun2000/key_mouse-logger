# key_mouse-logger

## 使用说明

- 首先运行create_log.py，会在同级目录下创建一个txt文件
- 然后运行其他的还没写的统计指标，可以看到各种信息


## 数据大小
20分钟-270KB    
1小时-810KB     
8小时-6480KB 约等于6.3MB        
一个月20天-126.56MB     
一年-1518.75MB  
一年的轨迹才这么点...
